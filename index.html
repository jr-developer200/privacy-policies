The permissions listed below may be requested by one or more of our apps depending on the app’s features. Only the permissions required by a specific app will be requested.

  • Location: ACCESS_FINE_LOCATION, ACCESS_COARSE_LOCATION, ACCESS_BACKGROUND_LOCATION — used for maps, location features, or location-based ads (background only if necessary).
  • Microphone: RECORD_AUDIO — used for voice input or audio recording.

  • SMS: SEND_SMS, RECEIVE_SMS, READ_SMS, RECEIVE_WAP_PUSH, RECEIVE_MMS — used only for SMS/messaging features or verification.
  • Storage: READ_EXTERNAL_STORAGE, WRITE_EXTERNAL_STORAGE — used to read/write files; newer Android uses scoped storage where possible.
  • Nearby / Bluetooth: BLUETOOTH_ADVERTISE, BLUETOOTH_CONNECT, BLUETOOTH_SCAN — used to discover/connect to nearby devices.
  • Notifications: POST_NOTIFICATIONS — used to show app notifications.
  • Draw over apps: SYSTEM_ALERT_WINDOW / display over other apps — used to show overlays or floating UI when required by the app.
  • Installed apps list: permission to access the list of installed apps (QUERY_ALL_PACKAGES or package visibility APIs) — used only when necessary to provide features like app integration or compatibility checks.

  • Exact alarms: SCHEDULE_EXACT_ALARM — used to schedule precise alarms or reminders when exact timing is required by app features (we request this only when necessary).
